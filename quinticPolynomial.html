Domkol: Quintic Polynomial

Function: Cubic Polynomial Quintic Polynomial Exponential

Explanation

This example is similar to the cubic polynomial example, expect it is quintic, i.e. the polynomial has five zeroes, so the 3D graph winds five times around the white circle if all the zeroes are inside the circle.

Compared to the cubic polynomial example, the canvas element is larger and there are more pixels per complex unit.
